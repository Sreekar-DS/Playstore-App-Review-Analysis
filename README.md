# Playstore-App-Review-Analysis
Data Analytics project featuring Exploratory Data Analysis (EDA) and Tableau Dashboard development on Google Play Store Apps and User Reviews using Python, Pandas, Seaborn, and Tableau to uncover factors influencing app success, ratings, installs, user engagement, and sentiment.

## Project Overview

This project explores Google Play Store application data and user reviews to identify the key factors influencing app success, user engagement, ratings, installs, and customer satisfaction. Using Exploratory Data Analysis (EDA), statistical insights, and interactive Tableau dashboards, the project uncovers patterns in app categories, pricing strategies, review sentiment, and installation trends.

The analysis was performed using Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn for data cleaning, transformation, and visualization. Additionally, an interactive Tableau dashboard was developed to provide business stakeholders with an intuitive view of app performance metrics and market trends.

## Business Objective

The objective of this project is to help app developers and businesses understand what drives success in the Android app ecosystem. By analyzing app characteristics and user feedback, the project provides actionable insights that can support product development, marketing strategies, and user engagement initiatives.
