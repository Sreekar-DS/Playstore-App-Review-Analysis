# Data Folder

## Overview

This folder contains the datasets used for the **Play Store App Review Analysis** project.

The datasets were used for Exploratory Data Analysis (EDA), data cleaning, visualization, and business intelligence dashboard development.

## Files

### 1. Play Store Data.csv

Contains information about applications available on the Google Play Store.

Key attributes include:

* App Name
* Category
* Rating
* Reviews
* Size
* Installs
* Type (Free/Paid)
* Price
* Content Rating
* Genres
* Last Updated
* Android Version

### 2. User Reviews.csv

Contains user-generated reviews for Play Store applications.

Key attributes include:

* App Name
* Translated Review
* Sentiment
* Sentiment Polarity
* Sentiment Subjectivity

## Purpose

These datasets were analyzed to identify:

* Factors influencing app success
* User engagement patterns
* Rating and installation trends
* Category-wise performance
* User sentiment and customer satisfaction

## Data Source

Google Play Store Apps and User Reviews Dataset (Educational Use)
